Game Development Showcase
-------------------------
This is a system designed to make the development of a game available to the public.

!!! This is a test version in prototype status and is not intended for official use. Responsible Use !!!

Prototype 0.0.1 Update 3
Peach Tree


Requirements
------------
PHP [8.2] oder upper


1. Upload the files to your project directory.
2. Set write permissions for the following folders.
   - data/
   - assets/
   - assets/uploads/
   - assets/devlog/
   - assets/landing/
   - assets/news/
   - assets/pages/
   - assets/screenshots/
   - assets/uploads/

3. Set write permissions for the following folders.
   - data/admin-auth.json
   - data/devlog-settings.json
   - data/devlog.json
   - data/footer.json
   - data/gallery.json
   - data/landing.json
   - data/news.json
   - data/pages.json
   - data/settings.json
   
4. Show your Site / Login To Admin
   - https://yourdomain.de
   - https://yourdomain.de/admin/edit.php
   
---
Standard Username for Admin: GameDevShow
Standard Password for Admin: GameDevShow


Please change at least the password!!
